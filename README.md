# Web Scraping Projects with Python
A collection of web scraping scripts and notebooks focused on extracting real-world data from  platforms like Amazon Egypt. 
This project demonstrates how to use automation and parsing libraries to build structured datasets.

---

# Tech Stack
- Requests: For send HTTP request easily and interact with websites.
- BeautifulSoup: For parsing HTML and extracting specific data points.
- Selenium: For browser automation and handling dynamic content.
